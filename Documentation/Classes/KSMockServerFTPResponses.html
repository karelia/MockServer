Inherits from NSObject

Overview

A collection of stock responses that can be used to fake an FTP server.

These responses cover many of the typical commands that an FTP server needs to implement.

retrResponse

Response to a RETR command, which is used to retrieve a file.

+ (NSArray *)retrResponse

Return Value

Array containing the regexp pattern to match against, and a list of commands that will be sent back to the client.

Discussion

Pretends to start listening for a client connection (actually the listener is already listening), then pauses for a bit and finally returns an indication that the data has been sent. This is fake, as the sending of the data isn’t synchronised with this response in any way, but the delay should ensure that the data actually has gone by the time the response is received.
