KSMockServerResponseCollection Class Reference

Inherits from NSObject
Declared in KSMockServerResponseCollection.h

Overview

Loads a group of named mock server responses from a JSON file, and allows you to obtain a responder using one of them.

Keeping the responses in JSON is a lot simpler than building up the array in code.
